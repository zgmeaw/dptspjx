# AW 多平台视频解析器

一个简洁、智能的多平台视频解析工具，支持抖音、快手、小红书等主流短视频平台。

## ✨ 核心功能

- 🎬 **多平台支持**：抖音、快手、小红书、皮皮虾、最右、微视等
- 🔄 **智能解析**：自动尝试多个API接口，确保解析成功
- 📱 **设备智能识别**：自动识别PC/移动端，提供最佳解决方案
- ⬇️ **智能下载**：根据设备类型提供不同的下载方案
- 🎨 **现代化UI**：渐变设计，流畅动画，响应式布局

## 🚀 使用说明

### 基础操作

1. **粘贴视频链接**（支持自动提取）
2. **点击解析视频**
3. **观看或下载视频**

### 💻 PC端使用

**视频播放：**
- 如果页面内无法播放（CORS限制），会自动提示
- 点击"在新窗口打开观看"按钮
- 在新窗口可以直接观看和下载

**下载视频：**
1. 点击"下载视频"按钮，视频会在新窗口打开
2. 在新窗口中右键点击视频 → "视频另存为"
3. 或点击播放器右下角菜单(⋮) → "下载"

### 📱 移动端使用

**视频播放：**
- 大多数情况下可以直接在页面播放

**下载视频（重要）：**

由于防盗链限制，需要使用专业下载工具：

1. **点击"下载视频"按钮**
2. **点击"复制视频链接"**
3. **使用下载工具粘贴链接**

**推荐的下载工具：**
- **Android**: ADM (Advanced Download Manager)、IDM+、迅雷
- **iOS**: Documents by Readdle、Alook浏览器

## ❓ 常见问题

### Q: PC端视频无法播放？
**A:** 这是CORS跨域限制，属于正常现象。
- ✅ 点击"在新窗口打开观看"按钮
- ✅ 在新窗口可以正常播放和下载

### Q: 移动端显示403错误？
**A:** 这是防盗链保护，无法通过网页绕过。
- ✅ 使用"复制链接"功能
- ✅ 在专业下载工具中粘贴链接
- ✅ 或返回原APP下载

### Q: 为什么不能直接下载？
**A:** 
- 视频平台有防盗链机制（检查Referer、User-Agent等）
- 浏览器有CORS安全策略
- 这些是服务器端的限制，纯前端无法绕过

**解决方案：**
- PC端：在新窗口打开，右键另存为
- 移动端：使用专业下载工具

### 为什么采用这种方案？

**技术限制：**
- 视频平台的防盗链机制在服务器端
- 浏览器的CORS策略无法通过JavaScript绕过
- 需要特定的请求头和认证信息

**最佳实践：**
- PC端：利用浏览器原生功能（新窗口打开）
- 移动端：推荐专业工具（可以绕过限制）
- 不依赖不稳定的黑科技方案
