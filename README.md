# AW 多平台视频解析器

一个简洁、强大的多平台视频解析工具，支持抖音、快手、小红书等主流短视频平台。

## ✨ 功能特性

- 🎬 **多平台支持**：抖音、快手、小红书、皮皮虾、最右、微视等
- 🔄 **智能解析**：自动尝试多个API接口，确保解析成功率
- 📱 **响应式设计**：完美适配PC端和移动端
- ⬇️ **智能下载**：支持视频下载，提供多个备用链接
- 🎨 **美观UI**：现代化渐变设计，流畅的动画效果

## 🚀 快速开始

### 在线使用

直接访问部署的网站，粘贴视频链接即可使用。

### 本地运行

```bash
# 克隆项目
git clone https://github.com/your-username/dptspjx.git

# 进入项目目录
cd dptspjx

# 使用任意HTTP服务器运行
# 例如使用Python
python -m http.server 8000

# 或使用Node.js的http-server
npx http-server
```

然后在浏览器中访问 `http://localhost:8000`

## 📖 使用说明

1. **复制视频链接**：在抖音等平台复制视频分享链接
2. **粘贴链接**：在输入框中粘贴链接（或点击"粘贴"按钮）
3. **解析视频**：点击"解析视频"按钮
4. **观看/下载**：在线观看或下载视频到本地

## ⚠️ 注意事项

### 视频播放问题说明

由于视频平台的防盗链机制和CORS限制，某些视频可能无法直接在浏览器中播放。这是正常现象，解决方案：

1. **使用下载功能**：将视频下载到本地观看（推荐）
2. **尝试备用链接**：系统会自动提供多个备用下载链接
3. **更换浏览器**：不同浏览器的安全策略不同

### API限制说明

本项目使用的第三方解析API可能存在以下限制：
- 请求频率限制
- 部分视频无法解析
- API服务可能不稳定

建议：合理使用，避免频繁请求
